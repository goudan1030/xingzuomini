# 云开发微信星座运势小程序开源版本

# 安装方法
1.下载到本地后，导入到微信开发者工具中，修改你自己的appid

2.开通云环境，新建“user” 数据库集合

3.点击链接，注册接口地址。https://www.showapi.com/from/17dd11c9360

4.修改你自己的showapi_appid 和 secret
